class xbmcaddon.Addon(id=None)[source]¶

Bases: object

Kodi’s addon class

Offers classes and functions that manipulate the add-on settings, information and localization.

Creates a new AddOn class.

Parameters:id – [opt] string - id of the addon as specified in addon.xml

Specifying the addon id is not needed. Important however is that the addon folder has the same name as the AddOn id provided in addon.xml. You can optionally specify the addon id from another installed addon to retrieve settings from it.**id** is optional as it will be auto detected for this add-on instance.

Example:

self.Addon = xbmcaddon.Addon()
self.Addon = xbmcaddon.Addon('script.foo.bar')
